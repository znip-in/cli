# znip
CLI to quickly add snippets to your project

## Installation

```bash
$ npm i -g znip
```

### Options v1.0.0

- [x] `znip snippet` - add snippet to project with same name as snippet

- [x] `znip snippet -n name` - add snippet to project with name
